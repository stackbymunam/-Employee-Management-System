# Employee Management System

## Overview
The Employee Management System is a modern web application designed to help organizations manage employee records and various HR-related tasks efficiently. Built with React and Vite for optimal performance.

## Features
- **Employee CRUD Operations** - Create, read, update, and delete employee records
- **Search & Filter** - Quickly find employees using multiple search criteria  
- **Responsive UI** - Clean, intuitive interface for seamless user experience
- **Fast Performance** - Built with Vite for instant hot module replacement
- **Code Quality** - Enforced with ESLint for consistent code standards

## Tech Stack
- **Frontend**: React 19.1.1, Vite 7.1.7
- **Styling**: Tailwind CSS 4.2.2
- **Build Tool**: Vite
- **Linting**: ESLint 9.36.0
- **Language**: JavaScript (Module-based)
